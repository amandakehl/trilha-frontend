## **Como baixar o projeto**

```bash
    # Para clonar o repositório 
    $ git clone https://github.com/your-username/trilhaFrontEnd 

    # Entrar no diretório 
    $ cd trilhaFrontEnd

    # Instalar as dependências 
    $ npm install 

    # Iniciar o projeto 
    $ npm start
```
